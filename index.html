PCIL-MVP: Perturbation Contrastive Imitation Learning with Multimodal Visual Perception for Active Visual Search

Yuhao Wang, Guohui Tian



Abstract

Active visual search (AVS) poses significant challenges for autonomous agents in unseen environments with only egocentric RGB-D observations. To address target recognition, semantic understanding, and localization under occlusions and uncertain layouts, we propose a multi-modal feature fusion framework for ground projection. Our method integrates RGB, semantic segmentation, and object detection features into a unified allocentric grid map, leveraging CLIP-based vision-language models and cross-attention mechanisms for semantic alignment. Reliable localization is ensured through a hybrid confidence estimation strategy that combines statistical analysis with learned predictions, enabling uncertainty-aware navigation decisions. Temporal dependencies are modeled via a Transformer-based multi-frame fusion module to enable resilient and coherent map updates in dynamic environments. Furthermore, we introduce a contrastive imitation learning paradigm incorporating adaptive sampling and perturbed trajectory augmentation to improve policy robustness and generalization. Extensive experiments show that our system outperforms baseline methods on visual localization and search tasks.
